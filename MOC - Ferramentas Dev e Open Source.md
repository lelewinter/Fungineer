# MOC — Ferramentas de Desenvolvimento e Open Source

> Mapa de conteúdo navegável. Use como ponto de entrada para explorar infraestrutura que torna IA extensível.

## O que é este tema

Padrão emergente: ferramentas open source, extensíveis e agnósticas a modelo vencem. MCP (Model Context Protocol) é padrão que conecta IA com qualquer ferramenta. Este cluster cobre MCP em domínios (ComfyUI image gen), indexação de codebase, eficiência de performance e tutoriais completos. Tema central é abertura > lock-in.
