# MOC — Modelos Locais e IA Privada

> Mapa de conteúdo navegável. Use como ponto de entrada para explorar IA que roda completamente no seu dispositivo.

## O que é este tema

A tendência local-first em IA permite que modelos, busca, visão por computador e processamento de áudio rodem no seu notebook ou telefone sem enviar dados para cloud. Este cluster aborda modelos destilados (Qwen), computer vision edge (MediaPipe, Micro-Handpose), TTS local (Mistral) e embeddings multimodais. A implicação estratégica é privacidade, performance, independência de API e controle total.
