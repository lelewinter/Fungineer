# MOC — Agentes Autônomos e Orquestração

> Mapa de conteúdo navegável. Use como ponto de entrada para explorar padrões emergentes de múltiplos agentes de IA trabalhando coordenados.

## O que é este tema

A próxima onda em IA é agentes múltiplos coordenados executando tarefas complexas sem intervenção humana. Este cluster aborda descoberta automática (Claude Peers), orquestração visual (Maestri), análise de inteligência pessoal (Crucix), indexação para eficiência (Indexação de Codebase) e aplicações ofensivas (Red Team autônomo). Essas notas exploram infraestrutura, padrões e implicações de agentes trabalhando juntos.
